# Go!

- 환경변수 GOPATH에 작업공간 경로를 추가한다. 

  나는 WORKSPACE/Go에서 작업할거라서 거기다 함

- src, bin, pkg 폴더를 만든다.

- src폴더 안에 main.go를 만든다.

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello World")
}
```

```
go run "c:\WorkSpace\Go\src\main.go"
```

이렇게 실행해도 되고 vscode 환경에서는 go extention으로 생성된 실행버튼을 클릭해도 된다.
