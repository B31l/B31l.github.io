# 네트워크 프로토콜

네트워크 프로토콜은 데이터의 전송이나 수신과 관련된 규칙을 의미한다. 서로 다른 네트워크에 연결

# OSI 참조 모델

전체 네트워크 시스템을 7개의 기능 계층으로 세분화하고, 각 계층이 정형화된 기능을 수행하도록 구성한 모델이다.

| 계층  |                 | 기능 |
| ----- | --------------- | ---- |
| 1계층 | 물리 계층       |      |
| 2계층 | 데이터링크 계층 |      |
| 3계층 | 네트워크 계층   |      |
| 4계층 | 전송 계층       |      |
| 5계층 | 세션 계층       |      |
| 6계층 | 표현 계층       |      |
| 7계층 | 응용 계층       |      |
