## 프로미스

프로미스는 이러이러한 값을 나중에 반환해주겠다~ 라고 **약속**하는 **객체**이다.

프로미스 객체는 다음과 같이 `new Promise`를 사용해 생성할 수 있다.

````js
const condition = true;

```
const promise = new Promise((resolve, reject) => {
  if (condition) {
    resolve("성공");
  } else {
    reject("실패");
  }
});
```
````

생성한 프로미스 객체 promise의 결괏값은 원하는 타이밍에 받으면 된다.
