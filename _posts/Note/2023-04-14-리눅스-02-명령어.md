# dd

### 날짜 및 시간 확인

```bash
$ date
```

### 시스템 정보 확인

```bash
$ hostname
```



```bash
$ uname
```



```bash
$ uname -a
```

### 사용자 정보 확인

```bash
$ whoami
```



```bash
$ who
```



### 디렉터리 내용 확인

```bash
$ ls
```
