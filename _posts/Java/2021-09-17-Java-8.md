# 상속

부모클래스는 자식클래스에 비해 개념적으로 넓다 (포괄적인 개념)<br>근데 기능적으로는 좁음

```
class 자식클래스 extends 부모클래스{
	//필드
	//생성자
	//메소드
}
```

다른 언어와는 다르게 자바는 다중 상속을 허용하지 않는다.



(부모 클래스는 자식 클래스를 포함하고 있으므로 형 변환이 가능하다.)

- 확장

- 재사용(reduce)  

- 관리용이
  - 여러 클래스에서 중복된 기능(행위)나 데이터(속성)을 하나의 클래스로 처리
