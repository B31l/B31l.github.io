# Loop

## while

```csharp
int i = 0;

while(i < 10) {
    
    if (i == 7) {
        i++;
        continue;
    }

    if (i == 9) {
        break;
    }

    if ((i % 2) > 0) {
        Console.WriteLine(i);
    }
    i++;
    
}
```

## do while

```csharp
string guess;

do {
    Console.WriteLine("Guess a Number");
    guess = Console.ReadLine();
} while (!guess.Equals("15"));
```
