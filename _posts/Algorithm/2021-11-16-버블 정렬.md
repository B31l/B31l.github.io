## Python[^1]

```python
from typing import 
def bubble_sort(arr: MutableSequence) -> None:
    n = len(arr)
    
    for i in range(n-1):
        for j in range(n-1, i, -1):
            
            if arr[j-1] > arr[j]:
                arr[j-1], arr[j] = arr[j], arr[j-1]
```

## Js

```js
function bubble_sort(arr) {
    const n = arr.length;
    
    for (let i=0; i<n-1; i++){
        for (let j=n-1; j>i; j--) {
            
            if (arr[j-1] > arr[j]) {
                [arr[j-1], arr[j]] = [arr[j], arr[j-1]];
            }
        }
    }
}
```
