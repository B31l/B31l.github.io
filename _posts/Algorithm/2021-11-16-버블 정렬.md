# 코드

```python
def bubble_sort(a: MutableSequence) -> None:
    
    n = len(a)
    for i in range(n-1):
        for j in range(n-1, i, -1):
            if a[j-1] > a[j]:
                a[j-1], a[j] = a[j], a[j-1]
```

# 시간 복잡도

|  B(n)  |  A(n)  |  W(n)  |
| :----: | :----: | :----: |
| O(n^2) | O(n^2) | O(n^2) |

$$
n개의  자료를 정렬할 때, {n(n-1) \over 2} 번의 비교가 발생한다.
$$
