# ✒

## 1. 설치

-   📌 [설치 링크](https://go.dev/dl/)

위 링크에서 운영체제와 버전을 확인하고 설치하시옹

기본 저장 위치인 **C:\Go\\**에 설치를 진행한다.

## 2. 환경변수 설정

고랭은 으앵 환경변수를 설정해야 해요

|             | 변수 이름 | 변수 값   |
| ----------- | --------- | --------- |
| 사용자 변수 | `GOPATH`  | C:\GoPath |
| 시스템 변수 | `GOROOT`  | C:\Go\    |

### 2.1. GOPATH

Go 프로젝트의 작업공간(workspace)

### 2.2 GOROOT

처음 Go가 설치된 폴더

### 2.3. GOBIN

`go install` 명령을 실행했을 때 바이너리 파일이 생성되는 디렉터리
