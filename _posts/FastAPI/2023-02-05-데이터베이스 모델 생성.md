# ORM

데이터베이스를 조작하기 위해서는 쿼리문을 사용해야 하는데, 이게 데이터베이스마다 사용하는 문법이 조금씩 달라서 일관적이지가 않음... 데이터베이스에 맞는 쿼리를 알아서 생성해주는 ORM을 사용하면 편하다.

ORM 라이브러리로는 `SQLAlchemy`를 사용한다.

```
pip install sqlalchemy
```
