# Bubble sort

> `버블 정렬`
>
> **끝으로 밀려납니다. 거품처럼...**
>
> 
>
> 인접한 두 값을 비교합니다.
>
> 조건을 만족한다면 교환합니다.

| B(n)  | A(n)  | W(n)  |
| :---: | :---: | :---: |
| O(n2) | O(n2) | O(n2) |

```python
def bubble_sort(a: MutableSequence) -> None:
    
    n = len(a)
    for i in range(n-1):
        for j in range(n-1, i, -1):
            if a[j-1] > a[j]:
                a[j-1], a[j] = a[j], a[j-1]
```

# Selection sort

> `선택 정렬`
>
> **ㅇㅇ**
>
> 
>
> ㅇ
>
> ㅇ

| B(n)  | A(n)  | W(n)  |
| :---: | :---: | :---: |
| O(n2) | O(n2) | O(n2) |

```python
def selection_sort(a: MutableSequence) -> None:
    
    n = len(a)
    for i in range(n - 1):
        min = i
        for j in range(i + 1, n):
            if a[j] < a[min]: min = j
        a[i], a[min] = a[min], a[i]
```
